Append docker-aliases.sh and docker-compose-aliases.sh to the end of ~/.bashrc

**Usage:**

*docker*

d - docker \
d r - docker run \
d e - docker exec \
d b - docker build \
d i - docker image \
d c - docker container \
d clr i - remove \<none\> images \
d clr c - remove unused containers  \

*docker-compose*

dc u - docker-compose up \
